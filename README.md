### 🧩 Enterprise AI Resume Generator
Multi-agent system that tailors resumes to job descriptions using retrieval-augmented generation.

**Architecture:** LangChain agent orchestration → ChromaDB vector store for semantic retrieval → FastAPI backend → LLM generation layer (OpenAI / Anthropic APIs)

**Key Engineering Decisions:**
- RAG pipeline grounds generation in the candidate's actual experience to reduce hallucinated claims
- ChromaDB vector storage for fast semantic matching between job descriptions and experience chunks
- FastAPI service layer for clean separation between orchestration and API surface

**Stack:** `LangChain` `ChromaDB` `FastAPI` `Python` `OpenAI API` `Anthropic API`

### 📈 Automated Cryptocurrency Trading Platform
Full-stack platform for automated trading strategy execution with real-time market data.

**Architecture:** React/TypeScript frontend ↔ FastAPI backend ↔ Redis for low-latency state/caching ↔ MongoDB for persistence, with live data via Binance API over WebSocket

**Key Engineering Decisions:**
- WebSocket connections for real-time price feeds instead of polling
- Redis layer to keep hot trading state out of the primary datastore
- Containerized with Docker and deployed on AWS for reproducible environments

**Stack:** `Python` `FastAPI` `Redis` `MongoDB` `React` `TypeScript` `WebSocket` `Binance API` `Docker` `AWS`
